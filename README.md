# 🌍 Overview

AgroInsight-Prediction is an end-to-end Machine Learning project that predicts suitable crops using soil measurements. It demonstrates preprocessing, EDA, feature engineering, model training, evaluation and comparison of multiple ML algorithms.

---

# ✨ Features

- Data preprocessing
- Missing value handling
- Exploratory Data Analysis
- Feature Scaling
- Label Encoding
- Random Forest
- Gradient Boosting
- Voting Classifier
- Logistic Regression
- Hyperparameter Tuning
- Model Comparison
